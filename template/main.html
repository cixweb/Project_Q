Introduzione

ProGuard è un tool implementato in Java, per effettuare lo shrink e l'offuscamento del bytecode Java. ProGuard riceve in input un file .jar e restituisce in output un nuovo file .jar più compatto, limitandone il reverse-engineering. Il file jar prodotto necessita di minori requisiti di memoria e può essere trasferito velocemente sulla rete. Le entità modellate dal sistema, quindi, riflettono le caratteristiche di un oggetto Java.
