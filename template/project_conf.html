Introduzione

La configurazione del progetto mantiene la stessa struttura in tutte le release:

  • docs:

    La documentazione completa, le licenze, il manuale e le FAQ. Tutto in formato html.

  • examples:

    File di configurazione di esempio.

  • lib:

    I file jars compilati e pronti per essere utilizzati.

  • src:

    Il codice sorgente.

Nella release 5.0 vengono aggiunte ulteriori directory:

  • bin:

    Scripts per eseguire Proguard, la GUI e ReTrace.

  • build:

    Scripts di build alternativi.

In ogni release viene aggiornata la documentazione e l'intera configurazione del progetto.
